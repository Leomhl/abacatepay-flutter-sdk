# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2024-11-28

### Added

- Initial release of AbacatePay Flutter SDK
- **Billing**: Create and list payment billings
- **Customer**: Create and list customers
- **Coupon**: Create and list discount coupons
- **PIX QR Code**: Create QR codes, check status, and simulate payments
- **Withdrawal**: Create, get, and list withdrawals
- **Store**: Get store information
- **Webhook**: Validate signatures and parse events
- Complete error handling with typed exceptions
- Full TypeScript-like type safety with Dart
